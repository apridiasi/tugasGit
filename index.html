Installation Laravel

1. Masuk Command Prompt Langkah pertama dalam install Laravel adalah masuk Command Prompt.


2. Masuk ke Folder XAMPP Sebelum melakukan instalasi Laravel, arahkan Command Prompt atau terminal menuju direktori file server.


3. Mulai Proses Install Laravel Selanjutnya jika sudah masuk direktori htdocs, Anda harus membuat request untuk mengambil (serta menginstall) file Laravel yang telah disediakan dalam repositori Github.


4. Cek Instalasi Laravel di Web Browser Setelah proses download file Laravel selesai, nantinya akan ada folder baru pada direktori file server dengan nama sesuai nama project yang telah Anda tentukan sebelumnya pada folder /xampp/htdocs.
